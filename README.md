# Inter-College Tournament Analysis

An interactive Streamlit application for analyzing inter-college tournament data with rich visualizations and insights.

## Features

- Interactive dashboards with multiple visualization types
- Real-time filtering of data
- Sport-wise analysis
- College and state-level insights
- Participant demographics analysis
- Sentiment analysis of feedback
- Image gallery with processing capabilities

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the application using:

```bash
streamlit run ETE3.py
```
